Deklaracja funkcji i function Expression
  1. // function Declaration
  2. function sayHello(name) {
  3. console.log('Hello, ', name);
  4. }
  6. // function Expression
  7. const sayHi = function sayHello2(name) {
  8. console.log(`Hi ${name}, how are You?`);
  9. }
  11. sayHello('Mark');
  12. sayHi('Michal');

W wtniku otrymujemy:

Hello, Mark

Hi Michal, how are You?

Różnica pomiędzy deklaracją funkcji a 'function Expression' polega na tym że przy deklaracji funkcji możemy ją wywołać w każdej chwili, nawed jeżeli deklaracja funkcji na samym końcu kodu. Przy zapisaniu funkcji do zmienej nie możemy korzystać z tej zmienej w kodzie ecześniej.

function

Ciekawostka: przy sprawdzeniu typu funkcji w wyniku mamy: funkcja, ale takiego typu danych w JS nie ma :)

W ogóle funkcja w JavaScript to jest objekt. Szczegóły można zobaczyć przy pomocy połecenia: console.dir(sayHello).

Poniżej przykład jak możemy wykorzystać anonimową funkcje:

  1. // No name function
  2. let counter = 0;
  3. const interval = setInterval(function(){
  4. if (counter === 5) {
  5. clearInterval(interval)
  6. } else {
  7. console.log(++counter);
  8. }
  9. }, 1000)

W ES6 pojawił się nowy sposób zapisywania funkcji (Arrow function):

  1. // Arrow functions. since ES6
  2. const arrow = (name) => {
  3. console.log('Hello, ', name);
  4. }
  6. const arrow2 = name => console.log(`Hi ${name}, how are You?`);
  8. arrow('Monika');
  9. arrow2('Ira');

Poniżej jeden ciekawy przykład jak możemy zamienić funkcje pow w bibliotece Math

  1. const pow2 = num => num ** 2;
  2. const powNum = (num1, num2) => {
  3. return num1 ** num2;
  4. }
  6. console.log(pow2(5))
  7. console.log(powNum(5, 3));

W standardzie EC6 pojawiły się parametry domyślne.

  1. // default paramets in functions
  2. const sum = (a, b) => a + b;
  3. const sum1 = (a, b = 1) => a + b;
  4. const sum2 = (a = 2, b = a + 1) => a + b;

W przypadku gdy nie podamy jeden z parametrów w funkcji sum to w wyniku otrzymamy NaN. Zeby zapobiec takiemu przypadkowie możemy ustalić parametry domyślne np. funkcja sum1 lub sum2.

W przypadku gdy chcemy skorzystać ze wszystkich parametrów ale nie wiemy ile tych parametrów funkcja posiada, to możemy skorzystać z takiej właściwości:

  1. function sumAll(...all){
  2. console.log(all);
  3. }
  5. sumAll(1,2,3,4,5,6,7,8,9);

Zmienne typu Number

Zmienne w JavaScript nie potrzebują deklaracji jakuego typu to jest liczba integer lub float

  1. const numberInt = 21; // integer
  2. const numberFloat = 47.3; // float
  3. const numberPow = 10e3; // 10e3 = 10000

Zmienne tego typu nałeżą no klasy Number. Klasa Number posiada następujące metody:

Number.MAX_SAFE_INTEGER
W ten sposób wyświetla się maksymalna wartość typu integer. Ona wynosi 9007199254740991. Ta liczba równa się 2 do 53 minus 1 (Math.pow(2,53)-1)
Number.MIN_SAFE_INTEGER
W ten sposób odzyskamy minimalną wartość integer -9007199254740991
Number.MAX_VALUE
1.7976931348623157e+308 - to jest najwieksza liczba w JS, która jest znacznie większa od maksymalnej wartości integer.
Number.MIN_VALUE
5e-324 to jest minimalna liczba w JS.
Number.POSITIVE_INFINITY
Infinity - czyli nieskończoność. Takiego samego wyniku możemy otrzymać dzieląc jaką kiłwiek liczbe na zero (np. 2 / 0 = Infinity)
Number.NEGATIVE_INFINITY
-Infinity. Nieskończonność ze znakiem minus
  1. console.log(Number.isFinite(Infinity));
  2. console.log(Number.isFinite(78));
W ten sposób można sprawdzić czy wartość nałeży do nieskończoności. W wyniku otrzymamy wartość typu Boolean. Infinity = false; 78 = true

Number.Nan

Not A Number. Ciekawostką jest to że typ tej wartości jest number :)

Wartość NaN możemy otrzymać w przypadku 2 / undefined.

  1. console.log(`NAN value (Not A Number). Number.NaN = ${Number.NaN}`);
  2. console.log(typeof NaN);
  3. const createNanValue = 2 / undefined;
  4. console.log(Number.isNaN(createNanValue));
  5. console.log(Number.isNaN(43));
  1. const stringInt = '40';
  2. console.log(stringInt + 2);
  3. console.log(Number.parseInt(stringInt) + 2);
  4. console.log(Number(stringInt) + 2);
  5. console.log(+stringInt + 2);

W przypadku gdy do zmiennej typu string dodamy zmienną typu integer - otzymamy typ string. W tym przypadku to będzie string '402'. W trzech kolejnych przypadkach otrzymamy zmienną typu number 42.

  1. const stringFloat = '40.42';
  2. console.log(parseFloat(stringFloat) + 2);
  3. console.log(+stringFloat + 2);

Podobna sytuacja będzie przy dannych typu float. UWAGA: jeżeli wykorzystamy parseInt - wtedy w wyniku otrzymamy liczbę bez części ułamkowej.

  1. // .toFixed
  2. console.log(0.4 + 0.2); // = 0.6000000000000001
  3. console.log((0.4 + 0.2).toFixed(6)); // 0.600000 - String
  4. console.log(parseFloat((0.4 + 0.2).toFixed)); // NaN
  5. console.log(+((0.4 + 0.2).toFixed(1))); // 0.6

Taki wynik otrzymujemy ponieważ JS wykonuje dodawanie tych liczb w systemie binarnym. Można to naprawić korzystając z metody .toFixed, ale mamy wynik jako String. Dlatego skorzystamy ze znanej nam metody parseFloat. W metodzie toFixed w nawiasach podajemy liczbę znaków po przycinku jaka musi zostać, jeśli tej wartości nie wpiszemy to otrzymamy NaN.

  1. // BigInt
  2. const bigIntValue = 9007199254740991424242n;
  3. console.log(`${typeof(bigIntValue)} = ${bigIntValue}`);
  4. console.log(bigIntValue - 1n); // = 9007199254740991424241n

Wartość typu BigInt wykorzystujemy wtedy gdy chcemy wykonać operacji na wartości większej od Number.MAX_SAFE_INTEGER, czyli 9007199254740991, lub wartości mniejszej od Number.MIN_SAFE_INTEGER. Dlatego przy inicializacji na końcu liczby wpisujemy literkę n.

Na liczbach typu BigInt możemy wykonywać operacji matematyczne.

Uwaga: nie możemy w BigInt wykorzystywać liczby zmienoprzycinkowe. Nie możemy używać BigInt z innymi typami Number.

Biblioteka Math

  1. // Math
  2. console.log(Math.E);
  3. console.log(Math.PI);

Biblioteka posiada konstanty E, PI które możemy wykorzystać dla naszych obliczeń.

  1. console.log(Math.sqrt(25)); // 5
  2. console.log(Math.pow(5, 3)); // 125
  3. console.log(Math.abs(-42)); // 42
  4. console.log(Math.max(12, -1, 46, 90, 128, 0)); // 128
  5. console.log(Math.min(12, -1, 46, 90, 128, 0)); // -1
  6. console.log(Math.floor(4.9)); // zaokrągla w dól = 4
  7. console.log(Math.ceil(4.2)); // zaokrągla do góry = 5
  8. console.log(Math.round(4.9)); // zwraca wartość zbliżoną do najblższej calej liczby
  9. console.log(Math.trunc(4.9)); // odcina wartości po przycinku
  10. console.log(Math.random());; // random number

Powyżej część metod które posida biblioteka Math

  1. function getRandomBetween(min,max) {
  2. return Math.floor(Math.random() * (max - min +1) + min)
  3. }
  4. const randomVar = getRandomBetween(10, 42);
  5. console.log(randomVar);

A to jest funkcja która zwraca wartość typu integer w zakresie między dwoma liczbami.
